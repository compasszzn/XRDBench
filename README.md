
# SimXRD-4M
## Benchmark
**Open Source:**  SimXRD-4M is freely available on our website (http://simxrd.caobin.asia/).

## Installation

To get started with XRDBench, you'll need to install the following libraries:

- PyTorch
- tqdm
- Weights & Biases (wandb)
- reformer-pytorch
- einops

## Run the Project
```
bash sh/run.sh
```
